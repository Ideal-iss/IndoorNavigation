# Bluetooth Навигация в Помещениях с помощью BLE-маячков

## Описание проекта
Данный проект реализует систему внутренней навигации в помещениях с использованием технологии Bluetooth Low Energy (BLE) и специальных маячков (beacons). Система предназначена для помощи пользователям в ориентировании в сложных внутренних пространствах, таких как торговые центры, больницы, аэропорты и офисные здания.

## Архитектура системы
Проект состоит из следующих компонентов:
- Мобильное приложение (клиент)
- Серверная часть (бэкенд)
- BLE-маячки (аппаратная часть)
- База данных

## Технологии
- Язык программирования: JavaScript/TypeScript
- Мобильная разработка: React Native / Flutter
- Серверная часть: Node.js / Python
- База данных: PostgreSQL / MongoDB
- Протокол: Bluetooth Low Energy (BLE)

## Структура проекта
```
project/
├── mobile/                 # Мобильное приложение
├── backend/               # Серверная часть
├── docs/                  # Документация
├── hardware/              # Описание аппаратной части
└── README.md
```

## Функциональные возможности
- Обнаружение BLE-маячков
- Расчет местоположения пользователя
- Построение маршрута до точки назначения
- Отображение карты помещения
- Навигация в реальном времени
