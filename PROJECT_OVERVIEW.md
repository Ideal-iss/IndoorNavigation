# Bluetooth Навигация в Помещениях с помощью BLE-маячков

## Описание проекта

Данный проект представляет собой систему внутренней навигации в помещениях с использованием технологии Bluetooth Low Energy (BLE) и специальных маячков (beacons). Система предназначена для помощи пользователям в ориентировании в сложных внутренних пространствах, таких как торговые центры, больницы, аэропорты и офисные здания.

## Архитектура системы

Проект состоит из следующих компонентов:

- **Мобильное приложение** (клиент): для взаимодействия с BLE-маячками и отображения навигации
- **Серверная часть** (бэкенд): для обработки данных и управления маршрутами
- **BLE-маячки** (аппаратная часть): для определения местоположения
- **База данных**: для хранения информации о помещениях, маячках и маршрутах

## Основные возможности

- Обнаружение BLE-маячков в радиусе действия
- Расчет местоположения пользователя с точностью до 1-3 метров
- Построение маршрута до точки назначения
- Отображение карты помещения
- Навигация в реальном времени

## Технологии

- **Язык программирования**: JavaScript/TypeScript
- **Мобильная разработка**: React Native
- **Серверная часть**: Node.js
- **База данных**: PostgreSQL
- **Протокол**: Bluetooth Low Energy (BLE)

## Установка и запуск

### Серверная часть

1. Перейдите в директорию `/backend`
2. Установите зависимости: `npm install`
3. Настройте переменные окружения в `.env` файле
4. Запустите сервер: `npm run dev`

### Мобильное приложение

Следуйте инструкциям в `/mobile/README.md`

## Требования к BLE-маячкам

- Поддержка Bluetooth 4.0 и выше
- Энергонезависимость (работа от батареек не менее 12 месяцев)
- Стабильность передаваемого сигнала
- Возможность настройки параметров (UUID, Major, Minor, TX Power)

## Точность определения местоположения

Система обеспечивает точность определения местоположения в пределах 1-3 метров за счет использования:

- Многомерной трилатерации с весовыми коэффициентами
- Комбинации фильтров (скользящее среднее + фильтр Калмана)
- Уточненной модели распространения BLE-сигналов в помещениях
