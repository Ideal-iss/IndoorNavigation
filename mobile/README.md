# Мобильное приложение для Bluetooth навигации в помещениях

## Общее описание

Мобильное приложение является клиентской частью системы Bluetooth навигации в помещениях. Приложение взаимодействует с BLE-маячками для определения местоположения пользователя и предоставляет навигационные функции внутри помещений.

## Архитектура приложения

### 1. Модуль BLE-сканирования
- Обнаружение BLE-маячков в радиусе действия
- Сбор данных RSSI от маячков
- Фильтрация и обработка полученных данных

### 2. Модуль локализации
- Вычисление текущего местоположения пользователя на основе данных от маячков
- Использование алгоритмов трилатерации и фильтрации

### 3. Модуль навигации
- Построение маршрута до заданной точки назначения
- Отображение текущего местоположения на карте
- Обновление маршрута в реальном времени

### 4. Модуль отображения
- Отображение карты помещения
- Визуализация текущего местоположения пользователя
- Отображение маршрута и навигационных инструкций

### 5. Модуль связи с сервером
- Отправка данных о местоположении (при необходимости)
- Получение информации о маршрутах и точках интереса
- Синхронизация данных с сервером

## Технологии

### Платформы
- Android (API 23+)
- iOS (iOS 10+)

### Фреймворки
- React Native (предпочтительно) или Flutter
- Для нативной разработки: Kotlin/Java для Android, Swift для iOS

### Библиотеки BLE
- react-native-ble-plx (для React Native)
- flutter_reactive_ble (для Flutter)

### Графика и карты
- React Native Maps или Mapbox для отображения карт
- SVG или Canvas для отображения внутренних помещений

## Основные функции

### 1. Сканирование BLE-маячков
- Непрерывное сканирование в фоновом режиме
- Энергоэффективный режим работы
- Обработка данных с нескольких маячков одновременно

### 2. Определение местоположения
- Вычисление координат на основе RSSI данных
- Фильтрация шума в сигнале
- Обновление местоположения с заданной частотой

### 3. Навигация
- Выбор точки назначения на карте
- Построение оптимального маршрута
- Отображение стрелки направления
- Уведомления о поворотах и ориентирах

### 4. Отображение информации
- Карта помещения с метками
- Текущее местоположение пользователя
- Информация о точках интереса
- Индикация направления движения

## Энергопотребление

- Минимизация энергопотребления при сканировании BLE
- Адаптивная частота сканирования в зависимости от активности
- Оптимизация работы в фоновом режиме

## Безопасность

- Шифрование передаваемых данных
- Анонимность данных о местоположении
- Соблюдение политики конфиденциальности
