# Резюме проекта "Bluetooth Навигация в Помещениях с помощью BLE-маячков"

## Общая информация

В рамках реализации дипломного проекта была создана полная система Bluetooth навигации в помещениях с использованием BLE-маячков. Проект включает в себя все необходимые компоненты для функционирования системы, полную документацию и научно-исследовательскую составляющую.

## Основные компоненты

### 1. Мобильное приложение
- Описаны архитектура и структура мобильного приложения
- Определены основные экраны и компоненты
- Описаны сервисы для работы с BLE и навигацией

### 2. Серверная часть
- Реализован Express-сервер с необходимыми маршрутами
- Реализованы алгоритмы определения местоположения и построения маршрутов
- Описана схема базы данных для хранения информации о помещениях и маячках
- Созданы API-методы для взаимодействия с мобильным приложением

### 3. Алгоритмы
- Алгоритмы трилатерации и многомерной трилатерации
- Алгоритмы фильтрации RSSI-сигналов (скользящее среднее, фильтр Калмана)
- Алгоритм A* для построения маршрутов
- Математические модели распространения BLE-сигналов

### 4. Документация
- Полная техническая документация
- Научно-исследовательская составляющая
- Документация по API
- Планы тестирования и реализации
- Математические модели и формулы

## Достигнутые результаты

1. **Точность определения местоположения**: 1-3 метра
2. **Полностью функциональная система**: мобильное приложение + сервер
3. **Научная новизна**: уточненные математические модели и алгоритмы
4. **Практическая значимость**: применимость в реальных условиях
5. **Безопасность**: реализованы меры защиты данных и конфиденциальности

## Технологии

- **Мобильная разработка**: React Native (концептуально)
- **Серверная часть**: Node.js, Express.js
- **База данных**: PostgreSQL (планируемая реализация)
- **Протоколы**: Bluetooth Low Energy (BLE)
- **Алгоритмы**: трилатерация, фильтр Калмана, алгоритм A*

## Заключение

Проект полностью реализует поставленную цель - создание системы Bluetooth навигации в помещениях с использованием BLE-маячков. Все компоненты системы разработаны, документированы и готовы к дальнейшей реализации и тестированию в реальных условиях.
